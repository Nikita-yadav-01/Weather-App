<h1>3D Weather Forecast Dashboard</h1>
<br>
<p>Author - Nikita Yadav</p>
<br>

<p><b>Technologies Used:</b> HTML5, CSS3, JavaScript (ES6), REST API Integration, WeatherAPI, Responsive Web Design.</p>
<br>
<p>Designed an interactive user experience with 3D animations, floating effects, hover interactions, and mobile-friendly layouts.</p>
<br>
<p>Integrated the WeatherAPI REST API to fetch and display real-time weather information based on user-entered locations.</p>
<br>
<ul>
<p><b>Features</b>
<li>Live weather data from WeatherAPI</li>
<li>User location/city input</li>
<li>Temperature display</li>
<li>Weather condition icon</li>
<li>Humidity and wind speed</li>
<li>Mobile responsive</li>
